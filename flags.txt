Quantum Mini Golf CLI Flags
===========================

Core Launch
-----------
--map <double_slit|single_slit|single_wall|no_obstacle> (default: double_slit)
    Choose the initial arena layout.

--mode <classical|quantum|mixed> (default: mixed)
    Set the initial display mode shown in the main window.

--course <quantum_demo|advanced_showcase> (default: none)
    Start in a predefined multi-stage showcase sequence.

--wave-profile <packet|front> (default: front)
    Select the initial wave packet shape used for quantum play.

--stop-mode <time|friction> (default: time)
    Decide how shots terminate: fixed duration or friction-based.

--wall-thickness <float> (default: 1.0)
    Scale the central barrier thickness multiplier.

--movement-speed <float> (default: 1.0)
    Multiply the ball and wave travel speed after each shot.

--shot-time <float> (default: 50.0; <=0 disables the limit)
    Cap simulation time per shot.

--res-scale <float> (default: 1.0)
    Upsample or downsample the simulation grid resolution.

--draw-every <int> (default: 3)
    Render every Nth simulation step.

--target-fps <float> (default: 30.0)
    Request a target refresh rate for the renderer.

--mouse-swing (default: enabled)
    Guarantee mouse swing input remains active even when a tracker is available.

--config-panel / --no-control-panel (default: show)
    Force the configuration panel window to open or remain hidden.


Tracker & Calibration
---------------------
--vr / --no-vr (default: hybrid; mouse swing enabled, tracker armed)
    Toggle between tracker-driven swings and mouse-only control.

--display-tracker / --no-display-tracker (default: show)
    Show or hide the OpenCV tracker debug window.

--calibration-path <file> (default: auto-discover)
    Load a specific calibration file instead of searching standard locations.

--skip-calibration-preview (default: false)
    Prevent the calibration snapshot viewer from opening at launch.

--calibrate-course (default: false)
    Run the manual corner-picking calibration helper before launching.

--calibrate-course-led (default: false)
    Run the LED auto-detection calibration helper before launching.


Gameplay & Physics
------------------
--sink-threshold <float> (default: 0.25)
    Probability threshold that marks the ball as holed.

--max-steps-per-shot <int> (default: 2048)
    Limit the number of physics steps processed per swing.

--quantum-measure / --no-quantum-measure (default: enabled)
    Enable or disable automatic quantum measurement events.

--measurement-gamma <float> (default: 1.0)
    Control the sharpness of the measurement sampling distribution.

--sink-rule <prob_threshold|measurement> (default: prob_threshold)
    Choose the hole detection strategy.

--edge-boundary <reflect|absorb> (default: reflect)
    Set behaviour for the arena boundary.

--boost-hole / --no-boost-hole (default: enabled)
    Toggle probability boosting toward the hole.

--boost-factor <float> (default: 0.10)
    Baseline boost factor used when boosting is active.

--boost-increment <float> (default: 0.08)
    Increment applied after each measurement when boosting is enabled.


Performance & Rendering
-----------------------
--perf-profile <quality|balanced|fast> (default: quality)
    Apply a preset bundle of performance flags.

--blit / --no-blit (default: off)
    Force matplotlib blitting on or off regardless of the preset.

--gpu-viz / --no-gpu-viz (default: off)
    Turn GPU-backed visualisation layers on or off.


Recording & Automation
----------------------
--record-video [PATH] (default: disabled)
    Render the scripted double-slit demo and exit instead of launching the UI.
    Provide PATH to override the output destination.

--record-output <PATH> (default: QuantumMinigolfDemo.mp4 when recording)
    Specify the MP4 file written by --record-video. Overrides the default path.

--headless (default: false)
    Use the non-interactive Agg backend and skip opening the Qt window.


Diagnostics & Backend
---------------------
--dump-config (default: false)
    Print the resolved GameConfig values before launching.

--backend <auto|cpu|gpu> (default: auto)
    Force the FFT backend through the QUANTUM_MINIGOLF_BACKEND override.
